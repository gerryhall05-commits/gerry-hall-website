Vercel deployment steps

1. Create a free account at vercel.com
2. Click Add New -> Project
3. Choose the option to import or upload this folder/project
4. Vercel should detect Vite automatically
5. Build command: npm run build
6. Output directory: dist
7. Click Deploy

After deployment, Vercel will give you a live website address ending in .vercel.app

Before deploying, edit src/App.jsx and replace:
- [Your Name]
- [your@email.com]
- [LinkedIn Profile URL]

Optional:
- add your headshot
- add your LinkedIn banner image
- connect a custom domain later
